Poker Chess

Game elements

  • A chess set
  • A deck of playing cards (55 cards)

Setup

The chess pieces are set up as for a normal game of chess, except that the board is turned by 90° so that the white player has h1 in his left corner. (The queens are placed on file 4.)

The deck is shuffled. Each player is dealt five cards, and the rest are placed in a pile, face-down.

Gameplay

The players take turns, starting with the white player. In each turn, a player can do one of two things:

Make a move:
The player chooses a card and plays it, face-up, into a common discard pile, and makes a chess move which corresponds to the card. Finally, the player draws a new card (replenishing to a 5-card hand), if available.
The meaning of the cards are as follows:
  • Black card: A pawn move.
    • Value 1-8: A move to the corresponding file.
    • Value 9: A move to an odd-numbered file.
    • Value 10: A move to an even-numbered file.
    • Value J, Q or K: A taking move.
  • Red card: A move with an non-pawn piece.
    • Value 1-8: A move to the corresponding file.
    • Value 9: A move with a rook.
    • Value 10: A move with a knight.
    • Value J: A move with a bishop.
    • Value Q: A move with a queen.
    • Value K: A move with a king.
  • Joker: A move with any threatened piece.
Discard cards:
The player shows two cards, discards both, and draws two new cards for replacement (if available). Both of the displayed cards must be unplayable (correspond to no valid move for the player in the current position), otherwise discarding is not an option.

Valid moves

The rules for which moves are valid are the same as in chess, except that it does not play any role whether a king is in check. This means that a player may make a move which puts his king in check or leaves him in check. Furthermore, a king may be taken (in which case the game ends; see "Game end").

Castling is counted as a king move: As far as the cards are concerned, it is a move with the king, to the square where the king ends up. It can be played with a joker if the king is initially threatened. The status of the rook does not matter.
Castling through a threatened square is allowed.

Game end

The game ends when either king is taken, or when the current player has no valid moves after the deck has run out (either because they have no cards left, or because they have a single unplayable card left).

Scoring

A player obtains points according to the pieces taken by the player. The points for each piece are the usual pawn values:
PiecePoints
King100
Queeen9
Rook5
Bishop3
Knight3
Pawn1
The player with the most points win.
